Problem

【AMPPZ2014】The Captain


Description

给定平面上的个点，定义到的费用为，求从号点走到号点的最小费用。

Input

第一行包含一个正整数()，表示点数。
接下来行，每行包含两个整数,()，依次表示每个点的坐标。

Output

一个整数，即最小费用。

Sample Input

Sample Output

标签：最短路

Solution

一道比较基础的最短路变形。
不难发现，在一个凸壳上，如果走任意一条弦，显然没有直接沿着凸壳走优。
所以分别按和排两次序，相邻点连边，可得到一个图。在此图上跑最短路即可。
注意，此题卡（大佬：不错啊，卡错误算法天经地义~）
